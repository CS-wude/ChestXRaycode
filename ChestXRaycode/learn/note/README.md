## 🎯 使用建议

### 学习顺序
1. **先理解理论** - 阅读混淆矩阵详解
2. **动手实践** - 运行项目中的评估代码
3. **分析结果** - 对照文档解读你的模型结果
4. **优化改进** - 基于分析结果改进模型

### 快速上手
```bash
# 1. 评估你的模型
python main.py eval --checkpoint checkpoints/best_model.pth

# 2. 对照文档理解结果
# 打开相应的.md文件查看详细解释

# 3. 运行演示脚本
python confusion_matrix_demo.py  # 混淆矩阵演示
python roc_demo.py              # ROC曲线演示
```
